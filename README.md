> This project is a proof of concept

PeerFile server powered by Node.js

## Setup

Clone this repository
```bash
git clone https://github.com/peerfile/peerfile-server.git cd peerfile-server
```

Install dependencies
```bash
npm install
```

Start the server
```bash
npm start
```
